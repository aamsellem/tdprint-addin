================================================================
TD PRINT VARIABLES - INSTALLATION EN 3 ETAPES
================================================================

ETAPE 1 : Ouvrir Word et l'editeur VBA
----------------------------------------
- Ouvrir Microsoft Word
- Appuyer sur Alt+F11


ETAPE 2 : Copier le code
----------------------------------------
- Menu : Insertion > Module
- Ouvrir le fichier INSTALL_COMPLET.bas avec un editeur texte
- Copier TOUT le contenu
- Coller dans le module VBA


ETAPE 3 : Executer l'installation
----------------------------------------
- Appuyer sur F5 (ou menu Execution > Executer)
- Attendre le message de confirmation
- Fermer Word et le rouvrir


UTILISATION
----------------------------------------
- Alt+F8 > TDPrint > Executer

Ou ajouter un bouton dans le ruban :
- Clic droit sur le ruban > Personnaliser
- Macros > TDPrint > Ajouter


FONCTIONNALITES
----------------------------------------
- 14 categories de variables
- Recherche en temps reel
- Favoris (sauvegardes automatiquement)
- Double-clic pour inserer
- Fonctionne 100% hors ligne


EN CAS D'ERREUR
----------------------------------------
Si l'erreur "Acces VBA refuse" apparait :

1. Fichier > Options
2. Centre de gestion de la confidentialite
3. Parametres du Centre de gestion...
4. Parametres des macros
5. Cocher "Accorder l'acces au modele objet
   des projets VBA"
6. OK et recommencer
